Bienvenue dans A4 Composer - Visual Companion

Nous allons co-concevoir ton application PWA mobile-first pour composer des images sur feuille A4

📋 Ce qui nous attend

  • Plan de travail A4 — Portrait/Paysage, 300 DPI, canvas HTML5
  • Import d'images — Captures d'écran partitions batterie, tailles variées
  • Manipulation tactile — Drag, resize, rotate, réordonnancement (z-index)
  • Export/Impression — PNG 300 DPI + impression sans fil Android
  • PWA complète — Manifest, Service Worker, installable, offline
  • Design Glassmorphism — Barre d'outils, Lucide Icons, thème clair/sombre

🎯 Stack technique imposée

Stack

  • HTML5 + CSS3 + Vanilla JS (pas de framework)
  • Canvas API 300 DPI (2480×3508 px portrait)
  • Lucide Icons via CDN (mapping nom→SVG)
  • Manifest.json + Service Worker minimal
  • 100% client-side, pas de backend

Contraintes

  • Mobile-first, tactile
  • Code modulaire, maintenable
  • Icônes Lucide → faciles à remplacer par SVG custom
  • PWA installable Android
  • Impression sans fil (Print API)
